# 23. 判断字符串以字母开头，后面字母数字下划线，长度6～30  [AQ](./00-question.md)

```javascript
/^[a-zA-Z]\w{5,29}$/
// 以字母开头
/^[a-zA-Z]/
//后面字母数字下划线
/^\w+$/
//长度6～30
/^{6,30}$/


// 邮政编码
/\d{6}/

// 小写英文字母
/^[a-z]+$/

// 英文字母
/^[a-zA-Z]+$/

// 日期格式 2019.12.1
/^\d{4}-\d{1,2}-\d{1,2}$/

// 用户名
/^[a-zA-Z]\w{5, 29}$/

// 简单的 IP 地址匹配
/\d+\.\d+\.\d+\.\d+/
```
